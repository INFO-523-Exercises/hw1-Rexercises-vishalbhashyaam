Base R doesn’t have a function for mode,

Creating a R function for mode, (works for unimodal, bimodal, multimodal data)

Mode <- function(x, na.rm=FALSE){
  if (na.rm) x<-x[!is.na(x)]
  ux <- unique(x)
  return(ux[which.max(tabulate(match(x,ux)))])
}
